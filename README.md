# Heat map
Python project to create heat maps using IMDB rating data.

# Example
![](images/silicon_valley.png)

## Specs:
  - Season by columns
  - Episodes by rows
  - Ratings in the table
  - Rating gradiant based on 0 (red) to 10 (green)

# Data sources
https://www.imdb.com/interfaces/
https://datasets.imdbws.com/
